# typescript + webpack + react + fis3

## 全局需要安装的环境

```
 nodejs
 
 npm install cnpm -g

 npm install webpack -g

 npm install typescript -g

 npm install tsd -g
 
```
## 开发完成构建工具 fis3 百度前端构建工具 （选装）

```
 npm install fis3 -g
 
```
## 使用

```
npm install 

tsd install 

```

## 运行 

```
 npm run dev 
 
```
